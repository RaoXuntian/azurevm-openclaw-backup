# azurevm-openclaw-backup

Private backup staging repo for selected OpenClaw files from this machine.

## Included
- workspace core docs (excluding memory/)
- custom restart-resume hook
- runtime/pending-resume.d README
- custom skills: daily-stock-analysis, no-card-search
- redacted OpenClaw config (`config/openclaw.redacted.json`)

## Excluded
- memory/ and other personal notes
- `.env` files
- device identity, pairing, approvals, pending devices
- unredacted API keys, passwords, and tokens
- git metadata from the source workspace

## Notes
This backup is intentionally sanitized for private GitHub storage.
